# SepsisXFair
Code and resources for the SepsisXFair project (sepsis prediction with XGBoost and SHAP).

## Dataset
The main dataset (Dataset.csv, ~146 MB) is hosted on Google Drive due to GitHub file size limits.
Download link: https://drive.google.com/file/d/1ip3taJCoowY8le6pyoIgfAeZApOi9X41/view?usp=sharing
After downloading, place "Dataset.csv" in the project directory before running the notebooks or colab.
